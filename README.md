time-approver
=============

node shell

### Usage:

1. Download this zip
2. Extract
3. Double click the atom.exe

### Configuration:

add a `.timeConfig.json` to your users home, i.e. "\Users\someguy" and add the employees you want to approve: 
```
{
  "email": "your@email.com",
  "password": "xxxxxxxxxx",
  "employees": [
     "Last, First",
     "Last, Second"
  ],
  "me": "Doe, John"
}
```
